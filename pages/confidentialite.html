Politique de Confidentialité

1. Responsable du traitement

Le traitement des données personnelles est effectué par la société Carbon, située à [adresse], joignable à l’adresse : [email support].

2. Données collectées

Nous collectons uniquement les données strictement nécessaires à l’analyse de votre empreinte carbone, notamment :

  • Données d’identification : nom, email.
  • Données issues d’API connectées : historiques de transport, achats, consommation énergétique, navigation, etc.
  • Données d’usage : statistiques d’utilisation de l’application.

3. Finalités de traitement

Les données sont traitées pour :

  • Analyser les émissions de CO2 liées à vos usages ;
  • Vous proposer des conseils personnalisés ;
  • Améliorer notre service (statistiques globales anonymisées) ;
  • Vous envoyer des notifications ou emails d'information (opt-in uniquement) ;
  • Afficher des contenus partenaires éco-responsables.

4. Base légale

Le traitement repose sur :

  • Votre consentement explicite (connexion aux API, envoi d’emails) ;
  • Notre intérêt légitime à améliorer le service ;
  • Le respect d’une obligation légale éventuelle.

5. Partage des données

Vos données ne sont jamais revendues. Elles peuvent être partagées uniquement avec :

  • Des partenaires techniques liés à l’hébergement et l’analyse des données ;
  • Des API tierces que vous avez explicitement connectées à votre compte ;
  • Des autorités compétentes uniquement en cas d’obligation légale.

6. Durée de conservation

Les données sont conservées tant que le compte est actif. À la suppression du compte, les données sont automatiquement effacées sous 30 jours (ou immédiatement si vous en faites la demande).

7. Hébergement des données

Les données sont hébergées dans l’Union Européenne sur des serveurs sécurisés (OVH, Scaleway, etc.).

8. Sécurité

Nous mettons en œuvre des mesures techniques et organisationnelles pour garantir la confidentialité et l’intégrité de vos données : chiffrement, audits de sécurité, accès restreints, etc.

9. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès à vos données ;
  • Rectification ;
  • Suppression ;
  • Portabilité ;
  • Limitation et opposition au traitement.

Vous pouvez exercer vos droits via l’adresse : [email DPO].

10. Cookies

Des cookies techniques et statistiques peuvent être utilisés. Vous pouvez gérer vos préférences via le bandeau de consentement.
